## Run locally

1. Install dependencies:
   npm install
2. Copy the example environment file:
   cp .env.example .env
3. Add your GitHub token with access to GitHub Models in `.env`:
   GITHUB_TOKEN=your_token_here
4. Start the profile site:
   npm start
5. Run the local model sample:
   npm run sample

The repository includes a small Node.js profile site and a separate sample that calls the GitHub Models inference endpoint. The sample automatically loads `.env` if it is present in the repo root.
